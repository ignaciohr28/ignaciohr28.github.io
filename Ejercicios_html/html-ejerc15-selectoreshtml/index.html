Tipos de selectores que entran en examen

Clase (class=””)

Este selector sirve para poder modificar todos los elementos que tienen asignada esa clase

Identificador (id=””)

Este elemento sirve para modificar las partes del documento que tengas seleccionadas con esa id.

Todos los elementos (*)

Este elemento sirve para modificar todo el documento, sin importar si los elementos estan seleccionados o no.

Varios elementos ( p, em, h1, h2)

Esto sirve para modificar todo lo que esta contenido dentro de estas categorias.

Elementos contenidos en otros elementos (ol li)

Esto sirve para modificar solo los elementos contenido dentro del que haya sido escogido previamento, un ejemplo seria el siguiente:

  1. modificado
    2. no modificado
  2. modificado
    3. no modificado

Sobrevuelo (strong:hover)

Este sirve para que se modifique el texto solo cuando pasas el cursor por encima del sitio previamente seleccionado.
